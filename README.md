## 命名规范
### 文件
* html 全部小写 index.html,user_info.html
* js 全部小写 news.js,page.news.js
* css 同js

### js
* 变量、函数名 小驼峰 hotWord,newsList,showTags()
* 控制器 大驼峰 NewsCtrl,NewsDetailCtrl
* 本地存储 全部小写 username,company_id,company_name

### html
* id 小驼峰 myTag
* class 连接符 my-tag pure-u-1-3
